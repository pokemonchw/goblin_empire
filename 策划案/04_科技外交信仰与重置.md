### 交易公式

```text
交易收益 = 基础出售量 * (1 + 贸易倍率 + 关系倍率 + 使馆等级倍率) * 随机宽度
关系变化 = 基础变化 + 政策修正 - 掠夺惩罚
```

### 掠夺规则

掠夺消耗军力和劳力，目标越强奖励越高。掠夺入口不属于研究卡片；当 `militaryPower` 资源已显示时，外交页追加独立掠夺区块。

```text
成功率 = clamp(5%, 95%, 50% + 军力优势 / 100)
掠夺收益 = 目标基础奖励 * 成功倍率 * (1 + 掠夺政策 + 战旗加成)
失败损失 = 劳力损失 + 受伤人口概率 + 关系下降
```

掠夺目标：

- 菌菇农洞：低风险，给菌菇和皮革。
- 商队营地：中风险，给金币和朽木。
- 矿盟哨站：中高风险，给铁矿、煤渣、钢锭。
- 地表村落：高风险，给战利品、金币、账册。
- 古代遗迹：后期目标，给魔晶、符文板、战利品和深渊倾向俘虏。

### 俘虏类型

俘虏是掠夺行动的特殊产物，也是哥布林帝国的差异化繁衍资源。系统只做抽象战略管理：玩家在获得俘虏后选择培育新生、洗脑改造或做成食物，不展示具体过程。当前实现中，俘虏类型决定苗床新哥布林的特质提示、属性偏置和初始技能偏置；俘虏质量决定处置收益倍率、逃脱风险和报复风险；苗床洗脑程度决定新生额外属性和孕育失败率。洗脑程度为 0 时不能培育新生；培育成功后洗脑程度降为结算前的 50%，失败后降为结算前的 10%。所有当前关押俘虏都会按单个俘虏 `0.03/s` 计入口粮菌菇消耗。

### 俘虏处置

| 处置方式 | 消耗 | 产出 | 风险 |
| --- | --- | --- | --- |
| 培育新生 | 不移除俘虏；将俘虏作为可重复使用的苗床；洗脑程度为 0 时不可执行 | 需要 1 个月孕育；成功时繁衍出 1 个 `origin = captive_bed` 的哥布林，应用俘虏类型属性偏置、`floor(洗脑程度 / 25)` 额外倾向属性和 `技能偏置 * 质量倍率`，随后洗脑程度降为结算前的 50%；失败时不生成哥布林，洗脑程度降为结算前的 10%；成功或失败后都需要 1 个月休养，休养完成且洗脑程度大于 0 时可再次培育 | 预览显示孕育 1 个月、休养 1 个月、失败概率 `max(10%, 45% - 洗脑程度 * 35%)`、洗脑程度、倾向属性加成、继承概率、质量逃脱风险和质量报复风险 |
| 洗脑改造 | 固定消耗 100 菌菇；不移除俘虏 | 提升 `brainwashLevel`，公式为 `round(12 * 质量倍率 * (1 + 洗脑棚改造倍率))`，上限 100；同时获得 `10 * 质量倍率 * (1 + 洗脑棚改造倍率)` 粗识；每座洗脑棚提供 +15% | 预览和俘虏详情浮窗显示洗脑固定消耗、洗脑提升、当前洗脑程度、孕育成功率、孕育失败率、质量逃脱风险和质量反抗/报复风险；孕育期间不可改造，休养期间仍可改造，菌菇不足时不可执行 |
| 做成食物 | 俘虏；移除该俘虏并终止其休养倒计时 | 获得 `40 * 质量倍率` 菌菇 | 预览显示服从波动 -2 到 +4，报复风险为质量报复风险 +5%；孕育期间不可执行，休养期间仍可执行；当前实现只写风险日志 |

当前实现的俘虏苗床是唯一常规人口增长入口，且不使用自然增长进度。洗脑程度为 0 的俘虏不能开始“培育新生”。苗床会让俘虏在“培育新生”后进入 `gestating`，一个月后按洗脑程度结算成功或失败；成功会生成新的哥布林，并把俘虏洗脑程度降为结算前的 50%；失败不会生成，并把俘虏洗脑程度降为结算前的 10%；无论结果如何都会进入 `resting` 休养一个月。休养完成后同一俘虏仍需洗脑程度大于 0 才能再次培育；休养期间不可再次培育，但仍可被洗脑改造或做成食物。新局开局自动拥有 1 个普通质量的村姑俘虏，确保玩家必须通过苗床繁育获得第一批新增哥布林；该俘虏在被做成食物或其他未来处置移除前，同样持续消耗菌菇口粮：

```text
newGoblin = createGoblin(origin = captive_bed)
newGoblin.attributes += captiveType.attributeBias + floor(brainwashLevel / 25)
newGoblin.skills[skillId] = captiveType.skillBias[skillId] * captiveQuality.multiplier
newGoblin.traits.push(captiveType.traitHint)
```

实现要求：每个新增哥布林必须写入稳定 `id`、姓名、出身 `origin = captive_bed`、属性、特质和初始技能；总人口由存活哥布林对象数量派生。

## 6. 威望重置

重置叙事：帝国过度扩张后旧地穴崩塌，幸存者带着传说、技术和宝藏迁徙到更深处。玩家失去大部分普通资源、建筑和人口，但获得“帝国遗产”。

帝国遗产获得建议：

```text
帝国遗产 = floor(sqrt(历史最高人口) + log10(历史总粗识 + 1) * 3 + 深渊门数量 * 5 + 成就奖励)
```

保留内容：

- 帝国遗产。
- 已购买的威望天赋。
- 部分成就。
- 挑战完成标记。

不保留内容：

- 普通建筑、普通资源、当前人口、当前科技。

## 7. 挑战模式

新存档或帝国迁徙后的未初始化新局属于游戏未开始状态。地穴正文此时只能显示“地穴控制台”和“新局模式”两个卡片；“新局模式”卡片内部同时提供“开始正常模式”和挑战选择入口，不能只显示挑战模式，也不能把每个挑战拆成独立页面卡片。选择正常模式后，`runMode` 记为 `normal`，本局不启用挑战惩罚；选择任一挑战后，`runMode` 记为 `challenge`，并写入对应 `activeChallengeId`。开局后地穴正文不再显示“新局模式”卡片，而是展开正式地穴循环。已有实质进度的旧普通存档加载时应迁移为正常模式，不能回到只显示挑战入口的未选择状态。

| 挑战 | 规则 | 奖励 |
| --- | --- | --- |
| 永冬地穴 | 菌菇产出长期 -50% | 食物建筑永久 +5% |
