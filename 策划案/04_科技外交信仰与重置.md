# 科技外交信仰与重置

## 菌菇寄生体与孢冠共生庭

所有菌菇寄生体只信仰母菌，`faithId` 固定为 `mother_fungus`。母菌信仰是菌丝网络归属，不是七神神灵血脉，因此菌菇寄生体的 `bloodlineId` 固定为 `null`、`bloodlinePurity` 固定为 `0`。

“菌菇寄生体”是智慧菌丝寄生其他智慧种族、接管神经与记忆后形成的新种族，主要活跃在地底。每名个体必须保留“原始所属种族”特质：运行时以 `raceId = fungus_parasite` 表示当前种族，以 `originalRaceId` 保存受寄生前的宿主种族；普通俘虏该字段为 `null`。数值上先完整保留原种族属性与技能，再叠加寄生体的 `狡黠 +2、感知 +2、力量 -1、敏捷 -1`，体现孢子记忆带来的智力增强和菌丝侵蚀造成的肉体弱化。苗床后代也继承这两层修正，界面必须分开显示。

新势力“孢冠共生庭”由母菌网络统治，以能够承载的孢子记忆划分阶层。它以菌菇和粗识交换魔晶；袭击任一节点都会引发菌网协同报复。

| 层级 | 地点 | 定位 | 俘虏品质 |
|---|---|---|---|
| 低 | 飘孢菌道 | 搬运腐殖、散播孢子和押送新宿主的外围菌道 | 普通 |
| 中 | 菌丝育成所 | 植入菌核、抄录宿主记忆的中层设施 | 熟练、精英 |
| 高 | 孢冠母巢 | 收藏历代宿主记忆的母菌统治节点 | 精英、传奇 |

三个地点均只产出菌菇寄生体，但会按地点抽取不同的原始所属种族。地点越高，宿主越稀有，守军、魔性收益和报复风险也越高。

从普通到传奇，每个品质固定两种俘虏，类型定义通过 `requiredQualityId` 与品质硬绑定：

| 品质 | 两种俘虏 | 核心倾向 |
|---|---|---|
| 普通 | 孢尘拾荒者、菌丝搬运体 | 采菌、搬运、服从 |
| 熟练 | 孢忆抄录者、菌脉医师 | 书写、仪式、制作、采集 |
| 精英 | 巢群谋士、孢甲护巢者 | 监督、书写、抢掠、仪式 |
| 传奇 | 孢冠先知、母菌化身 | 顶级仪式、书写、监督与采菌 |

## 3. 外交与掠夺

外交界面在玩家首次拥有 1 只存活哥布林后解锁，包含贸易、关系和使馆/据点等级；解锁外交时会自动解锁军工线的“大木棒”研究，完成大木棒后开放抢掠兵、训练坑和独立“掠夺”子标签。外交主界面下固定分为“外交”和“掠夺”两个子标签；两个子标签内先按“地底 / 地表 / 深渊”显示世界子标签，进入某个世界后再按所属势力显示势力子标签，避免所有世界和势力混在同一页面。地点列表使用与资源列表相同的紧凑行形式，行内显示核心数值、距离、在途数量和操作按钮，完整说明、成本、收益、风险、声名、距离、返程时间和缺口在鼠标悬停或键盘聚焦时以浮窗显示。弓弩不再作为掠夺前置，只保留为铁片阶段的军工扩展。俘虏处置保留在氏族页，不再作为外交页卡片。

### 外交声名资源

外交和掠夺共享两种声名资源，作为中后期目标门槛。

- 恶名：只能通过掠夺成功获得；掠夺失败会减少恶名。恶名越高，可掠夺越高级的地点。
- 善名：通过进行交易获得；掠夺成功会减少善名。善名越高，可和越高级的势力交易。
- 两者当前容量均为 100，不随常规生产自动增长，不作为普通价格字符串硬编码；所有增减由交易和掠夺结算写入资源状态。

### 交易公式

```text
交易收益 = 基础出售量 * (1 + 贸易倍率 + 关系倍率 + 使馆等级倍率) * 随机宽度
关系变化 = 基础变化 + 政策修正 - 掠夺惩罚
善名变化 = 阵营基础善名奖励
可交易 = 当前善名 >= 阵营善名门槛
交易返程时间 = 阵营距离秒数；出发时扣除成本，返程时结算收益、关系和善名
```

### 掠夺规则

掠夺由玩家通过与职业分配相同的按钮模式调整本次派出的战斗职业哥布林数量：`-` 减少 1 名，`+` 增加 1 名，`最大分配` 直接设为当前可派出的最大人数；最低值受目标最低队伍人数限制，最高值受当前可派出的抢掠兵和战争头目数量限制。掠夺不消耗劳力，也不消耗独立战斗资源；每名派出的战斗哥布林会立即消耗 100 菌菇，菌菇不足时不能发起。出发后按目标距离进入在途状态，返程前同一批战斗哥布林不能再次出击；成功、失败、伤亡、声名和俘虏都在返程时结算。目标越强奖励越高。掠夺入口不属于研究卡片；外交解锁后先开放“大木棒”研究，研究完成并解锁抢掠兵后，外交页开放独立“掠夺”子标签。

```text
掠夺成本 = 派出哥布林数量 * 100 菌菇
祖灵属性加成 = 祖灵数量 * 1%
队伍强度 = sum(派出哥布林个人掠夺强度，属性部分已计入祖灵全属性加成) * (1 + 建筑/装备/政策/契约 + 领袖出战加成)
强度优势 = 队伍强度 - 目标地点强度
成功率 = clamp(5%, 95%, 50% + 强度优势 / 100)
掠夺收益 = 目标基础奖励 * 成功倍率 * (1 + 掠夺政策 + 战旗加成)
伤亡概率 = clamp(3%, 55%, 22% - 强度优势 / 280 - 伤亡减免)
关系下降 = 目标基础关系惩罚 * clamp(75%, 150%, 1 + 强度优势 / 250) * 关系惩罚修正
报复概率 = clamp(0%, 75%, 目标基础关系惩罚 / 100 + max(0, -强度优势) / 200)
可掠夺 = 当前恶名 >= 目标恶名门槛
掠夺返程时间 = 目标距离秒数；出发时扣除行军口粮并锁定队伍，返程时按出发快照结算
掠夺成功：获得目标基础恶名，减少目标基础善名惩罚
掠夺失败：减少目标基础失败恶名惩罚
```

### 俘虏类型

俘虏是掠夺行动的特殊产物，也是哥布林帝国的差异化繁衍资源。系统只做抽象战略管理：玩家在获得俘虏后选择培育新生、洗脑改造或做成食物，不展示具体过程。当前实现中，俘虏类型决定苗床新哥布林的特质提示、属性偏置、初始技能偏置和基础年龄范围；俘虏质量决定处置收益倍率、逃脱风险、报复风险和基础寿命范围；苗床洗脑程度决定新生额外属性和孕育失败率。洗脑程度为 0 时不能培育新生；培育成功后洗脑程度降为结算前的 50%，失败后降为结算前的 10%。所有当前关押俘虏都会按单个俘虏 `3.125/s` 计入口粮菌菇消耗；无加成下 5 个菌菇床正好覆盖 1 个俘虏或哥布林口粮口数。战兽是掠夺成功后的额外捕获物，也属于人口口粮单位，但不占哥布林住房、不提供劳力；战兽需要先驯化，驯化后可作为不需要洗脑的苗床。

### 战兽

掠夺成功后，目标地点会按 `warbeastCaptureChance` 额外判定一次战兽捕获；捕获不会替代俘虏和资源收益。战兽保存为独立个体，字段包括 `speciesId`、`name`、`source`、`isTamed`、`tamingProgress`、`breedingState`、`gestationSecondsRemaining` 和 `restSecondsRemaining`。战兽作为人口单位消耗菌菇口粮，基础口粮为 `3.125/s * foodConsumptionRatio`；进入 `resting` 休养时口粮翻倍。断粮死亡顺序为俘虏优先，其次战兽，最后哥布林；战兽断粮死亡按屠宰收益回收菌菇。

### 战兽处置

| 处置方式 | 消耗 | 产出 | 风险 |
| --- | --- | --- | --- |
| 驯化 | 不移除战兽；每次增加 25 点 `tamingProgress`，达到 100 后 `isTamed = true` | 解锁该战兽的“作为苗床”入口；孕育中不可驯化 | 未驯化期间仍持续消耗菌菇 |
| 作为苗床 | 需要 `isTamed = true`、空闲且住房有空位；不需要洗脑 | 1 个月孕育后生成 1 个 `origin = warbeast_bed` 的哥布林，应用战兽物种属性偏置和特质提示；随后进入 1 个月 `resting` | 休养期间不可再次苗床，口粮翻倍；仍可屠宰 |
| 屠宰 | 可在任意非移除状态执行；移除战兽并终止孕育或休养 | 固定获得 100 菌菇 | 失去后续驯化和繁育价值 |

当前实现的俘虏苗床是唯一常规人口增长入口，且不使用自然增长进度。洗脑程度为 0 的俘虏不能开始“培育新生”。苗床会让俘虏在“培育新生”后进入 `gestating`，一个月后按洗脑程度结算成功或失败；成功会生成新的哥布林，并把俘虏洗脑程度降为结算前的 50%；失败不会生成，并把俘虏洗脑程度降为结算前的 10%；无论结果如何都会进入 `resting` 休养一个月。休养完成后同一俘虏仍需洗脑程度大于 0 才能再次培育；休养期间不可再次培育，但仍可被洗脑改造或做成食物。完成“公用苗床”后，每张俘虏卡显示单体“自动培育”按钮；开启后，系统只会在当前菌菇库存至少覆盖一秒现有口粮、该俘虏洗脑程度达到 100%、该俘虏不处于 `gestating` 或 `resting`、且住房有空位时，自动执行一次“培育新生”并进入正常孕育倒计时。多个俘虏同时满足自动培育条件时，按俘虏属性总值从高到低排序；当前实现的俘虏没有独立六维属性对象，因此属性总值按“俘虏类型属性偏置总和 * 质量倍率”估算，同分时保持当前俘虏列表先后顺序。新局开局自动拥有 1 个普通质量的村姑俘虏，确保玩家必须通过苗床繁育获得第一批新增哥布林；该俘虏在被做成食物或其他未来处置移除前，同样持续消耗菌菇口粮：

```text
newGoblin = createGoblin(origin = captive_bed)
newGoblin.attributes += captiveType.attributeBias + floor(brainwashLevel / 25)
newGoblin.skills[skillId] = captiveType.skillBias[skillId] * captiveQuality.multiplier
newGoblin.traits.push(captiveType.traitHint)
```

实现要求：每个新增哥布林必须写入稳定 `id`、姓名、出身 `origin = captive_bed`、属性、特质和初始技能；总人口由存活哥布林对象数量派生。

## 5. 祭祀与深渊系统

祭祀系统分两层：祖灵祭祀和深渊契约。

### 祖灵祭祀

资源：祖灵、祖灵回响。

祖灵与祖灵回响分工：

- 祖灵是氏族祖先数量。建成祖灵祭坛后，每次有哥布林自然老死，祖灵 +1；建成祭坛前的自然老死只记录统计和解锁条件，不追溯补发祖灵。
- 每个祖灵让所有战斗职业哥布林全属性 +1%。当前战斗职业为抢掠兵和战争头目；加成影响职业属性产出修正和掠夺个人强度中的属性部分。
- 祖灵回响是可消耗祭祀资源，来自巫医、祖灵祭坛、战利品献祭和特定节日事件，用于购买祖灵升级和单局倍率。

个体信仰规则：

- 哥布林在建立祖灵祭坛前没有信仰，`faithId = null`。
- 首次建成祖灵祭坛后，现有哥布林和之后出生的新哥布林默认信仰 `goblin_ancestor`，显示为“哥布林祖灵”。
- 战兽没有信仰，不保存 `faithId`，战兽卡片只显示“无信仰”作为说明。
- 俘虏保存 `faithId`。菌菇寄生体固定信仰 `mother_fungus`，不会无信仰或改信七神；其他新俘虏有概率无信仰，有信仰时从七神列表随机选择，但种族主信仰权重最高。

个体血脉规则：

- 血脉不是当前信仰，而是可遗传的神灵来源标记；字段为 `bloodlineId` 和 `bloodlinePurity`。
- 所有血脉都必须对应某个非祖灵神灵，例如沃海血脉源于 `fertile_sea`。没有神灵来源的个体保存 `bloodlineId = null`、`bloodlinePurity = 0`。
- 普通村姑俘虏和所有菌菇寄生体没有血脉；其他有七神信仰的俘虏按信仰反查同源血脉，并按品质随机纯度：普通 1%-10%，熟练 11%-35%，精英 36%-70%，传奇 71%-100%。母菌不属于七神，不存在母菌血脉。
- 哥布林血脉只能通过俘虏苗床从母体继承，不会因建立祖灵祭坛、改信祖灵、洗脑、天气或战兽苗床而变化。
- 血脉按纯度线性提供职业产出修正，并且必须同时结算权柄正面增幅和七宗罪负面影响。沃海血脉作为基准：100% 纯度时采菌工菌菇产出 +30%，工坊类职业因暴食副作用产出 -8%。
- 威望天赋“贪婪血脉”是长期遗产天赋名称，不等同于个体 `BloodlineDefinition`；后续若要把威望天赋转为个体血脉，必须新增迁移和命名拆分。
